Jesse's Summary of Basic HTML

HTML Structure
Every HTML document has to follow the same structure so that STUPID computers can understand them.
This skeleton structure is comprised of certain key elements. The beginning of an HTML document starts with !DOCTYPE HTML.
This is greyed out in text editors with syntax editing enabled as it is a void tag and just states the format of the document.
The next tag is the HTML tag which comes straight after the doctype. The last line of any html document will have the closing tags HTML.
This is followed by the HEAD tag which contains information about CSS and javascript files as well as the TITLE tag.
Then comes the BODY tag which is where all the contents of the web page is coded.

HTML Tags
There are many different types of HTML tags and remembering all of them would be a big task. That's why we can look them up with APIs and websites such as W3Schools.
We get opening tags and closing tags and both of them together form elements. Examples of these would be the B-tag or Em-tag.
Although there are exceptions such as void tags which do not need a closing tag. An example of a void tag would be the Break-tag.

Attributes
There are certain elements that can be given attributes.
Attributes such as href and src can be given values such as a link to another page on the internet or a link to an image to be displayed on your webpage.

Inline and Block Elements
HTML Elements are split into 2 categories: either Inline or Block elements.
Inline Elements keep the text within the opening and closing tags in the same line as the previous element just as if you were typing on a type writer .
Block Elements have an invisible "block" that surrounds the element. This block has width and height and forces the element to be displayed on a new line.
